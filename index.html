Stumblr

(Ruby on Rails, Backbone.js)

Stumblr is a full stack web application inspired by Tumblr.

The main functionalities include user authentication, creating posts with different types of file uploads based on Filepicker API, trending blogs and posts, reblogging, and following/liking capabilities. Notifications are ordered based on created time from a custom quicksort implementation.

Since most of the data is held in the posts, the sql queries were optimized by preloading tables to improve load and parsing of mass posts. The explore page has dynamically re-positioned stacking posts based on changing window and margin width.

Sunsteroid

(Javascript, HTML Canvas)

Sunsteroid is a computer version of the classic asteroids arcade game

In this game, the player is the sun, which must shoot all the asteroids coming its way. The player receives points from hitting an asteroid and loses health by getting hit by an asteroid. This game follows DRY practices based on a multi-level inheritance structure. Throughout the game the velocity of the asteroids are constantly re-calculated based on the norm vector and size of the image in a set interval.

Chess

(Ruby)

Colorized console-based full-featured two-player chess game

This is an implementation of the class game, Chess, implemented in Ruby/ Move validation algorithms are applied to each piece to enable full movement capabilities. Players are not allowed to move during another players turn and to an invalid spot. Other checks are kept in place to keep pieces inside the grid and for check/checkmate features. Highlighted spots also show a users possible positions.
